# Todo List App 

A simple and efficient **Todo List application** built using **HTML, CSS, and JavaScript** to help users manage daily tasks with ease and clarity.

##  Features
- Add new tasks instantly  
- Mark tasks as completed  
- Delete tasks with smooth UI interaction  
- Clean and minimal design  
- Responsive layout for all screen sizes  

##  Tech Stack
- HTML  
- CSS  
- JavaScript  


## What I Learned
- DOM manipulation and event handling in JavaScript  
- Managing dynamic lists efficiently  
- Improving UI/UX using CSS transitions  
- Writing clean and readable frontend code  

## Future Improvements
- Edit existing tasks  
- Save tasks using localStorage  
- Add task categories or priorities  
- Dark mode support
